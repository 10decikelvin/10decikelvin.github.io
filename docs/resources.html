Useful resources

As a student, I accumulate useful resources as I'm studying / coding / doing whatever and would love to share this with other people who might need it. Some resources are locked behind AES-GCM encryption for privacy reasons, and I promise it's properly secured so please only access stuff you are given access to.

AES-GCM encryptor

A browser-based file encryptor that encrypts and decrypts your files in line with the security standards set up by NIST. Use this as a much more secure alternative to 'passworded' zip files for data security.

French DELF B1/2 Vocab (with audio!)

1000+ somewhat commonly used french vocab that I manually compiled from reading comprehension activites, podcasts etc. Includes gender information for nouns and also audio files, in the form of an Anki deck.
